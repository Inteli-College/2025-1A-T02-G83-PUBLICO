## How it works

- Any board can act as a parent. Add children via the `+` next to a board or drag-and-drop to nest.
- Collapse/expand to focus your view; parents remember their state.
- Reorder parents and children to reflect current priorities.
- Nothing changes about access: existing workspace/board visibility rules continue to apply.

### Quick start
1. Create a Macro Topic (e.g., â€œEnd of Year Campaignâ€).
2. Click the `+` beside it to add sub-boards like â€œBlack Fridayâ€ and â€œChristmasâ€.
3. Drag any existing board under a parent to re-parent it.
4. Collapse groups to keep your sidebar tidy.
